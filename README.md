# Bulls and cows game

Simple solution play Bulls&Cows game in terminal made with NodeJS

### Installation

App requires [Node.js](https://nodejs.org/) v8+ to run.

```sh
$ cd BullsAndCows
$ npm install 
$ node app.js
```
